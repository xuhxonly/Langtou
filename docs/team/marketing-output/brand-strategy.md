## 一、品牌核心价值主张

### 1.1 品牌使命
**"让每一种热爱都被看见，让每一次分享都有回响。"**

榔头致力于打造一个真实、温暖、有品质感的内容社区，让年轻人在分享生活的同时，发现志同道合的伙伴，实现从"记录"到"连接"再到"价值创造"的完整闭环。

### 1.2 品牌愿景
成为中国年轻人最信赖的生活方式分享平台，3年内进入内容社区赛道TOP5。

### 1.3 核心价值主张（Value Proposition）

| 维度 | 核心主张 | 用户价值 |
|------|----------|----------|
| **真实感** | "真实生活，不打滤镜" | 拒绝过度包装，鼓励真实记录，降低创作门槛 |
| **发现感** | "你的热爱，值得被算法偏爱" | XGBoost智能推荐，越用越懂你的兴趣图谱 |
| **成长感** | "从分享到变现，每一步都算数" | 创作者成长体系+带货橱窗，让热爱可持续 |
| **安全感** | "清朗社区，青少年友好" | 青少年模式+反作弊，打造健康社区环境 |
| **效率感** | "AI帮你，创作不卡壳" | AI标题/标签/文案/封面，降低创作阻力 |

### 1.4 品牌定位金字塔

```
                    [情感共鸣]
              "你的生活，值得被看见"
                         |
            [功能利益] "AI赋能的智能创作社区"
                         |
        [产品属性] "图文视频+智能推荐+创作者变现"
                         |
              [品类归属] "生活方式内容社区"
```

### 1.5 品牌人格（Brand Persona）

| 维度 | 定义 | 表现方式 |
|------|------|----------|
| 年龄 | 25岁 | 年轻有活力，但不幼稚 |
| 性别 | 中性偏女性 | 包容多元，尊重差异 |
| 性格 | 真诚、好奇、有品位、乐于助人 | 内容审核严格但沟通亲切 |
| 口头禅 | "试试呗" "这也太会了吧" "学到了" | 年轻化表达，避免说教 |
| 社交角色 | 那个总能给你安利好东西的朋友 | 种草而非推销 |

## 二、品牌Slogan候选方案

### 2.1 候选方案一览

| 编号 | Slogan | 创意阐释 | 适用场景 |
|------|--------|----------|----------|
| **S1** | **"榔头一下，生活有料"** | 双关"榔头"品牌名，"有料"既指内容丰富又指生活精彩 | 主推Slogan，全渠道通用 |
| **S2** | **"真实生活，值得分享"** | 直击小红书"滤镜焦虑"痛点，强调真实性差异化 | 品牌TVC、开屏广告 |

## 三、品牌视觉调性

### 3.1 品牌色彩体系

| 色彩角色 | 色值 | 使用场景 | 心理暗示 |
|----------|------|----------|----------|
| **主色-榔头橙** | `#FF6B35` | Logo、主按钮、重点强调 | 活力、热情、行动力 |
| **辅色-暖石灰** | `#2D3436` | 标题文字、深色背景 | 沉稳、品质、高级感 |
| **点缀色-薄荷青** | `#00D2D3` | 标签、图标、进度条 | 清新、年轻、科技感 |
| **背景色-奶油白** | `#FFF9F5` | 页面底色、卡片背景 | 温暖、舒适、亲和力 |
| **功能色-成功绿** | `#00B894` | 成功状态、正向反馈 | 积极、健康、可信 |
| **功能色-警示红** | `#FF4757` | 错误提示、删除操作 | 醒目、紧急 |

### 3.2 色彩应用比例

```
60% 奶油白（背景/留白）
30% 暖石灰（文字/结构）
7%  榔头橙（CTA/重点）
3%  薄荷青（点缀/装饰）
```

### 3.3 品牌字体规范

| 场景 | 中文字体 | 英文字体 | 字重 |
|------|----------|----------|------|
| 品牌Logo | 思源黑体 Heavy | Montserrat Bold | 900 |
| 大标题 | 思源黑体 Bold | Montserrat SemiBold | 700 |
| 正文 | 思源黑体 Regular | Inter Regular | 400 |
| 辅助说明 | 思源黑体 Light | Inter Light | 300 |
| 数字/数据 | DIN Alternate Bold | DIN Alternate | 700 |

### 3.4 视觉风格关键词

- **摄影风格**：自然光、生活化场景、微距细节、真实肤色
- **插画风格**：扁平几何、2.5D微立体、品牌橙+薄荷青渐变
- **排版风格**：大留白、模块化卡片、圆角16px、微阴影
- **动效风格**：弹性缓动、微交互反馈、页面转场流畅自然

### 3.5 视觉禁忌

- 禁止使用过度磨皮/美白的人像素材
- 禁止使用高饱和荧光色（除品牌橙外）
- 禁止使用尖锐直角（统一圆角处理）
- 禁止使用密集信息堆砌（保持呼吸感）

## 四、目标用户画像

### 4.1 核心人群A：都市生活探索者（占比40%）

| 维度 | 描述 |
|------|------|
| **基础信息** | 22-28岁，一二线城市，本科以上学历，月收入8K-20K |
| **生活状态** | 独居或合租，周末喜欢探店/看展/短途旅行 |
| **内容偏好** | 美食探店、穿搭分享、家居改造、旅行攻略 |
| **消费特征** | 愿意为"氛围感"和"体验感"付费，注重性价比 |
| **使用场景** | 午休刷Feed、睡前种草、周末规划出行 |
| **痛点需求** | 厌倦了过度包装的"精致生活"，渴望真实推荐 |
| **平台行为** | 高频浏览，中频互动，低频创作（月发布1-3篇） |
| **获取渠道** | 小红书迁移用户、抖音内容引流、应用商店搜索 |

### 4.2 核心人群B：兴趣圈层深耕者（占比35%）

| 维度 | 描述 |
|------|------|
| **基础信息** | 20-30岁，一二线城市，垂直兴趣领域爱好者 |
| **生活状态** | 有明确的兴趣标签（摄影/手账/健身/数码/美妆等） |
| **内容偏好** | 深度测评、教程干货、装备清单、进阶技巧 |
| **消费特征** | 为兴趣舍得投入，追求专业度和 authenticity |
| **使用场景** | 学习技能、寻找同好、展示作品、获取认可 |
| **痛点需求** | 现有平台内容同质化严重，算法推荐不够精准 |
| **平台行为** | 中高频创作，高频互动，愿意经营个人IP |
| **获取渠道** | 垂直社区引流、KOL推荐、搜索引擎长尾词 |

### 4.3 核心人群C：潜力创作者（占比25%）

| 维度 | 描述 |
|------|------|
| **基础信息** | 24-35岁，有专业技能或独特生活经验 |
| **生活状态** | 希望将知识/经验变现，但缺乏平台和工具 |
| **内容偏好** | 专业知识分享、经验总结、带货种草、课程推广 |
| **消费特征** | 关注ROI，需要清晰的变现路径和数据反馈 |
| **使用场景** | 内容创作、粉丝运营、商品上架、数据分析 |
| **痛点需求** | 平台抽成高、变现门槛高、缺乏创作辅助工具 |
| **平台行为** | 高频创作，注重数据，积极尝试变现功能 |
| **获取渠道** | 创作者大会、行业社群、竞品平台迁移 |

### 4.4 用户画像对比矩阵

| 维度 | 都市生活探索者 | 兴趣圈层深耕者 | 潜力创作者 |
|------|---------------|---------------|-----------|
| 年龄 | 22-28 | 20-30 | 24-35 |
| 性别占比 | 女70% / 男30% | 女55% / 男45% | 女50% / 男50% |
| 城市分布 | 一线/新一线 | 一线/二线 | 一线/新一线 |
| 日均使用时长 | 45-60min | 60-90min | 90-120min |
| 内容消费:创作比 | 9:1 | 6:4 | 4:6 |
| ARPU值（月） | ¥50-100 | ¥80-150 | ¥200-500 |
| 核心诉求 | 发现+种草 | 学习+社交 | 成长+变现 |
