## 一、增长战略总览

### 1.1 增长北极星指标

**"月活跃创作者数（MAC）"**

选择理由：创作者数量直接驱动内容供给，内容供给驱动用户消费，用户消费驱动创作者变现意愿，形成正向飞轮。

### 1.2 辅助增长指标

| 指标层级 | 指标名称 | 定义 | 当前值 | 3个月目标 | 6个月目标 | 12个月目标 |
|----------|----------|------|--------|-----------|-----------|------------|
| 一级 | DAU | 日活跃用户数 | 0 | 50万 | 200万 | 800万 |
| 一级 | MAU | 月活跃用户数 | 0 | 200万 | 800万 | 3000万 |
| 二级 | 新增激活 | 首次打开App用户 | 0 | 100万 | 400万 | 1500万 |
| 二级 | 次日留存 | 次日回访率 | - | 45% | 50% | 55% |
| 二级 | 7日留存 | 7日回访率 | - | 30% | 35% | 40% |
| 二级 | 30日留存 | 30日回访率 | - | 18% | 22% | 28% |
| 三级 | 内容发布率 | 发布过内容的用户占比 | - | 15% | 20% | 25% |
| 三级 | 人均使用时长 | 日均使用分钟数 | - | 35min | 45min | 55min |
| 三级 | 人均消费内容数 | 日均浏览/播放内容数 | - | 80 | 100 | 120 |

### 1.3 增长飞轮模型

```
        +------------------+
        |   创作者增长      |
        |  (招募+激励+变现) |
        +--------+---------+
                 |
                 v
        +------------------+
        |   内容供给增加    |
        |  (图文/视频/直播) |
        +--------+---------+
                 |
                 v
        +------------------+
        |   用户体验提升    |
        |  (推荐+互动+社区) |
        +--------+---------+
                 |
                 v
        +------------------+

        |  (拉新+激活+留存) |
        +--------+---------+
                 |
                 +----------> (变现反哺创作者)
```

#### 2.3.2 热点借势机制

**热点响应SOP：**
1. **监测**（T+0）：舆情系统监测全网热点，30分钟内预警
2. **研判**（T+1h）：评估热点与品牌关联度，决策是否跟进
3. **创作**（T+3h）：内容团队产出热点关联内容
4. **发布**（T+6h）：全渠道同步发布，配合投放加热
5. **复盘**（T+3d）：数据复盘，沉淀热点内容模板

## 三、激活策略（Activation）

### 3.1 新用户引导流程（Onboarding）

#### 3.1.1 首次启动体验

```
[启动页] -> [兴趣选择] -> [内容偏好测试] -> [推荐关注] -> [首条内容消费] -> [创作引导]
   |            |              |                 |              |              |
 2s品牌        3-5个兴趣       5道选择题         10个推荐账号    沉浸式Feed     拍摄引导
 动画          标签（多选）     （生活方式/       （可跳过）      （自动播放）    （可选跳过）
                              美妆/美食等）
```

**关键设计原则：**
- 总时长控制在90秒内，每步可跳过
- 兴趣选择直接影响首次Feed内容，确保"第一眼心动"
- 推荐关注列表包含50%头部创作者+50%潜力创作者

#### 3.1.2 新用户任务体系（7日任务）

| 天数 | 任务名称 | 任务内容 | 奖励 | 设计目的 |
|------|----------|----------|------|----------|
| Day 1 | "初次见面" | 完成注册+完善资料 | 100积分+专属头像框 | 建立基础画像 |
| Day 2 | "发现美好" | 浏览10条内容并点赞3条 | 50积分+流量包 | 培养消费习惯 |
| Day 3 | "勇敢表达" | 发布第一条内容 | 200积分+新人推荐位 | 跨越创作门槛 |
| Day 4 | "找到同好" | 关注5个感兴趣的用户 | 50积分+关注列表优化 | 建立社交关系 |
| Day 5 | "深度互动" | 评论3条内容+收藏2条 | 50积分+收藏夹扩容 | 提升互动深度 |
| Day 6 | "分享快乐" | 分享1条内容到微信 | 100积分+分享专属贴纸 | 触发社交裂变 |
| Day 7 | "一周达人" | 累计获得10个赞 | 300积分+"新星创作者"徽章 | 正向反馈闭环 |

**完成全部7日任务额外奖励：** ¥20现金红包（可提现）

### 3.2 首内容发布激励

#### 3.2.1 "零门槛创作"计划

| 激励项 | 规则 | 价值 |
|--------|------|------|
| AI创作助手 | 新用户前3次发布可使用AI标题/标签/文案 | 降低创作阻力 |
| 新人流量包 | 首条内容自动获得500曝光推荐 | 确保首次正向反馈 |
| 模板创作 | 提供50+热门内容模板，一键套用 | 降低创作门槛 |
| 拍摄引导 | 首次拍摄提供实时构图/光线提示 | 提升内容质量 |
| 即时反馈 | 首条内容24小时内必获官方账号评论 | 情感激励 |

#### 3.2.2 首内容发布漏斗优化

```
注册完成用户 (100%)
    |
    v
点击发布按钮 (45%)  <- 优化：FAB按钮动效引导
    |
    v
完成内容编辑 (25%)  <- 优化：AI助手+模板+简化编辑器
    |
    v
成功发布内容 (18%)  <- 优化：发布流程<3步
    |
    v
获得首个互动 (12%)  <- 优化：新人流量包+官方互动
    |
    v
7日内再次发布 (8%)  <- 优化：发布奖励+创作挑战
```

**优化目标：将"成功发布内容"转化率从18%提升至30%。**

### 3.3 Aha Moment设计

**定义榔头Aha Moment："新用户在注册后24小时内，消费5条以上内容并发布1条内容，且获得至少1个互动。"**

| 触点 | 策略 | 预期提升 |
|------|------|----------|
| 兴趣选择页 | 根据选择实时预览"你的Feed将长这样" | 兴趣完成率+15% |
| 首次Feed | 前5条内容100%匹配兴趣标签 | 内容消费率+20% |
| 内容消费中 | 第3条内容时弹出"你也来分享吧" | 发布转化率+10% |
| 发布成功后 | 即时推送"已有XX人看到你的内容" | 留存率+12% |
| 获得互动时 | 推送通知+App内红点强提醒 | 回访率+25% |
