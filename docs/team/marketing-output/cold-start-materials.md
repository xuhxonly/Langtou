## 一、品牌物料清单

### 1.1 App图标设计需求

| 维度 | 规范要求 |
|------|---------|
| **设计理念** | "榔头"品牌具象化——简约几何化的榔头图形，搭配品牌橙主色，传递"敲开真实生活"的品牌理念 |
| **主色调** | 榔头橙 `#FF6B35` 占70%，暖石灰 `#2D3436` 占25%，薄荷青 `#00D2D3` 点缀5% |
| **风格** | 扁平化设计，圆角处理（外圆角12px），微立体阴影（2px offset），现代简约 |
| **图形元素** | 简约榔头轮廓+对话气泡/分享图标融合，暗示"分享"和"连接" |

#### 尺寸规范

| 用途 | 尺寸（px） | 格式 | 备注 |
|------|-----------|------|------|
| App Store图标 | 1024x1024 | PNG | 无圆角，系统自动裁切 |
| Android自适应图标 | 108x108（前景）/ 432x432（背景） | PNG | 遵循Material Design规范 |
| 应用商店展示 | 512x512 | PNG | 各商店展示用 |
| 网站Favicon | 32x32 / 16x16 | ICO/PNG | 网站标签页图标 |
| 推送通知图标 | 24x24 | PNG | Android通知栏 |
| PWA图标 | 192x192 / 512x512 | PNG | 渐进式Web应用 |
| 社交媒体头像 | 400x400 | PNG | 各平台官方账号头像 |

#### 色彩禁忌
- 禁止使用高饱和荧光色（除品牌橙外）
- 禁止使用渐变色超过3种
- 禁止使用黑色纯底（用暖石灰替代）
- 禁止使用竞品相似配色（避免小红书红、抖音黑）

