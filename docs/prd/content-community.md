## 1. 产品概述

### 1.1 产品定位
榔头是一款 **AI 知识闯关社区**。内容社区作为"知识闯关"的素材生产与消费载体，参考小红书"短平快、图文并茂、话题聚合"的产品形态，通过 UGC（用户生成内容）沉淀优质内容，并作为 AI 闯关题库的来源。

### 1.2 核心价值
- 对用户：记录学习心得 / 分享解题思路 / 通过消费他人内容获得启发
- 对平台：沉淀领域知识内容库，为 AI 闯关提供高质量训练素材
- 商业化：品牌话题挑战赛、信息流推广位

### 1.3 关联服务
| 微服务 | 职责 |
|--------|------|
| `langtou-content-service` | 笔记CRUD、标签、话题、Feed聚合 |
| `langtou-interact-service` | 点赞、评论、收藏、分享 |
| `langtou-user-service` | 用户、关注关系、粉丝 |
| `langtou-message-service` | 私信、站内信、通知 |
| `langtou-ai-service` | AI 摘要、标签推荐、一键生成闯关 |
| `langtou-ad-service` | 信息流广告、品牌话题挑战赛 |
| `langtou-quiz-service` | 闯关玩法（由笔记一键生成 Quiz） |

## 3. 详细需求

### 3.1 内容发布

#### 3.1.1 动态图文笔记（P0）

**用户故事**  
作为一名注册用户，我希望发布带图文的动态笔记，以便分享我的学习心得和解题思路。

**验收标准**
1. 用户点击底部 "+" 按钮进入发布页，默认进入"图文"模式。
2. 支持上传 **1–9 张图片**，首图作为封面；图片支持拖拽排序、单张删除。
3. 支持填写标题（≤ 40 字）、正文（≤ 2000 字），正文支持换行、表情、**富文本（加粗/斜体/代码块）**。
4. 正文中以 `#` 开头、空格或结尾结束的字符串自动识别为话题标签，点击可跳话题聚合页。
5. 支持手动添加话题标签（最多 5 个），输入时给出模糊匹配建议。
6. 支持选择"发布地点"（可跳过）。
7. 可见性：公开 / 仅粉丝 / 仅自己。
8. 点击"发布"后，1 秒内返回成功反馈，笔记进入审核队列（默认机器审核，敏感词命中再人工复审）。
9. 未登录用户点击发布按钮强制跳转登录。

**核心流程**  
```
用户 → 点击"+" → 进入发布页 → 选图/排序 → 填写标题+正文+话题 → 选择可见性 → 点击发布
 → 客户端校验 → 上传图片到 OSS → 调用 ContentService.createNote
 → 触发 SensitiveWord 校验 → 写入 notes 表 → 返回笔记ID → Toast"发布成功"
```

#### 3.1.2 视频笔记（P0）

**用户故事**  
作为一名用户，我希望发布短视频笔记（最长 5 分钟），用视频形式讲解我的解题过程。

**验收标准**
1. 支持从相册选择或拍摄视频，**最大 5 分钟**，支持格式 mp4 / mov，分辨率 ≥ 720p。
2. 上传过程中显示进度条，支持断点续传（基于分片上传）。
3. 发布前支持**封面裁剪**：自动截取视频任意帧作为封面，或从本地上传。
4. 填写标题、正文、话题标签，与图文笔记规则一致。
5. 视频经过 **CDN 转码**（生成多清晰度：360p/720p/1080p）。
6. 审核流程同图文笔记，额外校验视频时长、分辨率。
7. 发布成功后，笔记在 Feed 中首屏以视频卡片形式展示（含播放图标、时长）。
